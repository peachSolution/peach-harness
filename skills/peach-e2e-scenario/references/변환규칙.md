# 변환 규칙

Playwright codegen 녹화 코드를 CDP 시나리오로 변환할 때 적용하는 규칙.

## 1. 브라우저 연결 변환

```javascript
// ❌ 녹화 코드 (새 브라우저 실행)
const browser = await chromium.launch();
const context = await browser.newContext();
const page = await context.newPage();

// ✅ 변환 후 (CDP 연결)
const { connect } = require('../../lib/connect');
const { page, origin } = await connect();
```

> `require` 경로는 시나리오 파일 위치에 따라 조정한다. `lib/connect.js`까지의 상대경로.

## 2. URL 변환

CDP에서 상대경로가 동작하지 않는다. origin을 추출하여 절대경로로 변환한다.

```javascript
// ❌ 녹화 코드
await page.goto('http://localhost:3000/some/path');

// ✅ 변환 후 (안전 기본값: 자동 이동 금지)
if (!page.url().includes('/some/path')) {
  throw new Error('예상 경로가 아님 — 사용자 확인 후 다시 실행하세요.');
}

// ✅ 비민감 세션 + 사용자 실행 승인 확보 시에만 자동 이동 (SPA)
if (!page.url().includes('/some/path')) {
  await page.goto(`${origin}/some/path`);
  await page.waitForURL('**/some/path');
}

// ✅ 비민감 세션 + 사용자 실행 승인 확보 시에만 자동 이동 (MPA)
if (!page.url().includes('/some/path')) {
  await page.goto(`${origin}/some/path`);
  await page.waitForLoadState('networkidle');
}
```

> 민감 세션, 로그인 유지 탭, 관리자/결제 화면은 위 자동 이동 예시를 그대로 복사하지 않는다.
> 먼저 현재 탭 상태를 보고하고, 사용자 승인 후에만 `page.goto()`를 허용한다.
> SPA(Vue/React/Next.js)는 `networkidle` 대신 `waitForURL()` 또는 특정 요소 대기를 사용한다.
> MPA(PHP/Django/Rails 등)는 `waitForLoadState('networkidle')` 사용.

## 3. 브라우저 종료 제거

CDP 모드에서는 브라우저를 닫지 않는다.

```javascript
// ❌ 제거
await browser.close();
await context.close();
```

## 4. 디버그 코드 제거

```javascript
// ❌ 제거
await page.pause();
```

## 5. 다이얼로그 핸들러 (`setDialogHandler` 교체 + 원복 필수)

alert/confirm 다이얼로그를 자동 수락한다.
CDP에서는 dialog 핸들러가 세션에 잔존하므로 **반드시 finally에서 원복**한다.

```javascript
// ❌ page.on('dialog') 직접 등록 — 기본 handler와 충돌 가능
page.on('dialog', async (dialog) => {
  await dialog.accept();
});

// ✅ setDialogHandler로 교체 → finally에서 원복
const { connect, setDialogHandler } = require('../../lib/connect');
const { page, defaultDialogHandler } = await connect();

const dialogHandler = setDialogHandler(page, defaultDialogHandler, async (dialog) => {
  console.log(`📢 다이얼로그: ${dialog.message()}`);
  try { await dialog.accept(); } catch (_) {}
});

// ... try 블록 ...

finally {
  setDialogHandler(page, dialogHandler, defaultDialogHandler);
  process.exit(exitCode);
}
```

## 6. 대기 방식 변환

프레임워크에 따라 적절한 대기 방식을 선택한다.

### SPA (Vue/React/Next.js — 클라이언트 라우팅)

```javascript
// ❌ 고정 대기
await page.waitForTimeout(3000);

// ✅ URL 변경 대기
await page.waitForURL('**/step2');

// ✅ 특정 요소 대기
await page.locator('[data-testid="step2-content"]').waitFor({ timeout: 5000 });
await page.getByRole('heading', { name: '제목' }).waitFor({ timeout: 5000 });
```

### MPA (PHP/Django/Rails — 풀 리로드)

```javascript
// ✅ 네트워크 유휴 대기
await page.waitForLoadState('networkidle');
```

단, 짧은 대기(200~500ms)는 UI 애니메이션/전환 효과 용도로 허용한다.

## 7. 셀렉터 전략 (프레임워크 공통)

### 모던 프레임워크 (React/Vue/Next.js)

```javascript
// 버튼 — getByRole 우선
await page.getByRole('button', { name: '저장' }).click();

// 입력 필드
await page.getByLabel('이름').fill('홍길동');
await page.getByPlaceholder('검색어 입력').fill('검색어');

// data-testid (가장 안정적)
await page.getByTestId('submit-button').click();

// 텍스트로 찾기
await page.getByText('완료').click();
```

### 레거시 프레임워크 (PHP/jQuery)

```javascript
// id/class 기반
await page.locator('#customer_name').fill('홍길동');
await page.locator('a', { hasText: '검색' }).click();

// 테이블 내 링크
await page.locator('table tbody td').getByRole('link', { name: '관리' }).first().click();
```

> **주의**: CSS-in-JS 해시 클래스 (`.sc-xxxxx` 등) 직접 사용 금지 — 빌드마다 변경됨

## 8. 모달 처리

프레임워크에 따라 모달 구조가 다르다. `references/프레임워크-대응.md` 참조.

### 모던 프레임워크 (React Portal / Vue Teleport)

```javascript
// iframe 없이 DOM에 직접 렌더링
const modal = page.getByRole('dialog');
await modal.waitFor({ timeout: 10000 });
await modal.getByRole('button', { name: '확인' }).click();
```

### 레거시 프레임워크 (iframe 모달)

```javascript
// iframe 내부 접근
const frame = page.frameLocator('iframe[src*="target"]');
await frame.locator('a', { hasText: '실행' }).click();
```

## 9. 페이지 이동 대기

### SPA (클라이언트 라우팅)

```javascript
await page.getByRole('button', { name: '다음' }).click();
await page.waitForURL('**/step2', { timeout: 5000 });
```

### MPA (풀 리로드)

```javascript
await page.locator('a', { hasText: '다음' }).click();
await page.waitForLoadState('networkidle');
```

## 10. Select / Input 이벤트

```javascript
// select
await page.getByRole('combobox', { name: '상태' }).selectOption('완료');

// input fill
await page.getByLabel('수량').fill('3');

// 이벤트 수동 발생이 필요한 경우 (Vue v-model, React controlled input 등)
await page.locator('#field').evaluate(el => {
  el.dispatchEvent(new Event('input', { bubbles: true }));
  el.dispatchEvent(new Event('change', { bubbles: true }));
});
```

## 11. 셀렉터 디버깅

시나리오 실행 중 셀렉터 에러가 발생하면 agent-browser eval로 현재 DOM을 확인한다.

```bash
# 셀렉터 존재 여부 확인
agent-browser eval "document.querySelector('[role=dialog]') !== null"

# 버튼 텍스트 목록 확인
agent-browser eval "JSON.stringify(Array.from(document.querySelectorAll('button')).map(function(b){return b.innerText}))"

# 현재 URL 확인
agent-browser eval "location.pathname"
```

> iframe 내부 디버깅은 agent-browser로 불가 → playwright-cli fallback:
> ```bash
> ./e2e/pwc.sh eval "document.querySelector('iframe[src*=target]').contentDocument.querySelector('#element').innerText"
> ```
